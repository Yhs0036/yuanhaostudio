# AI 使用说明

## 使用了哪些 AI 工具

- Codex：用于需求拆解、简历信息提炼、React + Vite 项目搭建、页面代码生成、样式迭代和构建验证。

## AI 帮助完成了哪些工作

- 从简历 PDF 中提取个人信息、教育背景、项目经历和技能标签。
- 将简历内容改写为更适合个人网站展示的模块文案。
- 创建 React + Vite 项目结构，包括首页 Hero、精选项目、能力模块和整屏联系方式。
- 生成暗色科技感的视频背景资源，并接入到首页 Hero。
- 补充运行说明、构建脚本、依赖配置和基础项目文档。

## 自己手动修改了哪些内容

- 对页面要求、整体视觉方向、模块顺序和展示重点进行了确认。
- 后续会根据参考截图和目标网站继续调整视觉风格、项目内容和线上部署信息。

## 遇到的问题以及如何解决

- 简历 PDF 文件名包含中文，初次读取时出现编码问题；通过扫描桌面 PDF 并设置 UTF-8 输出解决。
- pnpm 默认拦截 `esbuild` 构建脚本；通过 `pnpm approve-builds --all` 明确允许 Vite 所需依赖完成安装。
- 内置浏览器预览过程曾被中断；最终通过本地 Vite 服务和 `pnpm build` 验证基础版本可运行、可构建。
